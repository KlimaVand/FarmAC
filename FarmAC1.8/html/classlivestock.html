◆ CalcMaximumProduction()

bool livestock.CalcMaximumProduction ( )
inline

Calculates daily production permitted by energy available.

Production of milk or growth is first calculated from energy available. Then there is a check to see if this production can be supported by the available N and if not, the maximum production is reduced.

Returns
bool which is false if there is an error.

energyIntake is in MJ per animal per year,

if parameter file says that weight loss is expected (e.g. in lactating dairy cows in early lactation)
